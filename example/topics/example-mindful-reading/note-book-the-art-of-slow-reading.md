# 《慢读之道》读书笔记（示例）

> ⓘ 本笔记内容**完全虚构**，书名、作者、引文均为演示而编造，不对应任何真实出版物。
> 它展示的是 `note-book-*.md` 笔记应有的结构（frontmatter + 摘录 + 核心观点 + 疑问 + 关联）。

## 摘录

> "我们以为自己在读书，其实常常只是在让眼睛经过文字。真正的阅读发生在你抬起头、合上书、试着复述的那一刻。"

> "注意力像预算：你没法同时支付给十件事。决定不读什么，和决定读什么同样重要。"

## 核心观点

- **阅读 = 输入 + 复述**：作者把"合上书复述一遍"当作阅读的必要环节，而非可选项。
- **注意力预算**：把注意力视为有限资源，主张主动"削减输入"而非追求"读得更多"。
- **三句话笔记法**：每读完一个章节，强制写三句话——讲了什么、我信不信、它和什么有关。

## 疑问

- "复述"对虚构类文学是否同样有效？还是更适合非虚构？
- 注意力预算的比喻会不会过度工具化阅读，反而削弱沉浸式体验？
